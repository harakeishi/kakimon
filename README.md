# kakimon（カキモン）

> 君と育つモンスター — 勉強で育てる学習アプリ

子供向けの学習PWA。学習コンテンツ（数字・ひらがな・漢字の書き取りなど）に取り組むと
ゲーム内通貨が貯まり、モンスターの餌や装備品を購入して一緒に育てていく。
学習コンテンツはプラグインとして追加できるよう設計されている。
